# BrightTV
Analysis of a TV Channel subscription using existing data will assist the CEO in growing the subscription base for this financial year. The analysis was conducted using a spreadsheet dataset that was recorded for a period of three (3) months.

The objective was to anlyse the existing data to develop ways to increase usage and consumption from viewership point of view.

Data Source: the dataset was in a spreadsheet formart that was divided into user profile and and viewership.

Tools that were used to analyse the data include, Excel spreadsheets, SQL, Miro, Canva and powerbi.

The insights are as follows:

Bright TV, is operating across all South African provinces with a focus on user consumption patterns.
Data indicates that male viewers and senior citizens represent the primary audience, while sports and local entertainment drive the highest engagement. 
Regional activity is most concentrated in Gauteng and the Western Cape, with viewership peaking during weekday afternoons and weekend sporting events. 
To expand its market share, the report suggests targeting younger demographics through after-school programming and culturally relevant local content. 
Ultimately, the source outlines a growth plan centered on content diversification and referral incentives to boost the platform's overall reach.
